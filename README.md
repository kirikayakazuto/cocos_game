使用creator + node.js 制作的游戏
===

用户中心
---
用户中心主要用于用户的个人资料修改,好友互动,任务奖励领取,好友以及世界排行
* 个人信息系统
    * 修改个人信息
    * 游客账号升级 -> 绑定手机号码
    * 注销账号(未完成)
* 好友系统
    * 添加好友
    * 验证好友请求(未完成)
    * 拉黑好友(未完成)
* 排行榜系统
    * 世界排行榜
    * 好友排行榜
* 任务系统
    * 用户任务, 每日签到
    * 邮件系统, 领取系统奖励
* 系统公告
    * 最新活动

游戏中心
---
游戏中心主要用于玩家进行游戏, 可以在线匹配玩家, 也可以创建房间(房卡模式)
* 五子棋
* 斗地主
    * 创建房间
    * 加入房间
* 牛牛
